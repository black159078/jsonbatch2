=> How to use Vite

1. Install 
    npm install --save-dev vite

2. Create a new file as vite.config.js

3. Update package.json
    "scripts": {
        "dev": "vite",
        "build": "vite build"
    }


4. Run Build the app for production
    npm run build

5. Run Vite in development server
    npm run dev
